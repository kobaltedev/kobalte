# v0.3.0

**January 14, 2023**.

## Breaking changes

- `Alert` root component is now `Alert.Root`.
- `Button` root component is now `Button.Root`.
- `Checkbox` root component is now `Checkbox.Root`.
- `ContextMenu` root component is now `ContextMenu.Root`.
- `Dialog` root component is now `Dialog.Root`.
- `DropdownMenu` root component is now `DropdownMenu.Root`.
- `HoverCard` root component is now `HoverCard.Root`.
- `Link` root component is now `Link.Root`.
- `Popover` root component is now `Popover.Root`.
- `RadioGroup` root component is now `RadioGroup.Root`.
- `Select` root component is now `Select.Root`.
- `Select.Multi` component is now `MultiSelect.Root`.
- `Select.Value` selected value render prop is now `selectedValue` and is of type `Accessor<string>`.
- `Separator` root component is now `Separator.Root`.
- `Switch` root component is now `Switch.Root`.
- `Tabs` root component is now `Tabs.Root`.
- `TextField` root component is now `TextField.Root`.
- `ToggleButton` root component is now `ToggleButton.Root`.

## New features

- Added `MultiSelect.*` components equivalent to `Select.*` components.
